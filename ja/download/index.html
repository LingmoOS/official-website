システム要件

最高の体験を得るためには、デバイスが以下の要件を満たしていることを確認してください。

プロセッサ

  • • 最小要件：Intel/AMDデュアルコアプロセッサ
  • • 推奨構成：Intel Core i3/AMD Ryzen 3以上
  • • x86_64アーキテクチャのみサポート

メモリ

  • • 最小要件：2 GB RAM
  • • 推奨構成：4 GB RAM以上
  • • SWAPパーティションを有効化することをお勧めします

ストレージ容量

  • • 最小要件：10 GBの空き容量
  • • 推奨構成：30 GB以上の空き容量
  • • ext4、btrfsファイルシステムをサポート

グラフィックカード

  • • Intel HD Graphics 4000以上をサポート
  • • NVIDIAグラフィックカードをサポート（オープンソースドライバーをお勧め）
  • • AMD Radeonグラフィックカードをサポート

ネットワーク

  • • インストールと更新にはインターネット接続が必要です
  • • 10 Mbps以上のブロードバンド接続をお勧めします
  • • Wi-Fiと有線ネットワーク接続をサポート

その他の要件

  • • UEFIファirmwareサポート（推奨）
  • • TPM 2.0セキュリティチップ（オプション）

デバイスが上記の要件を満たしていることを確認してください

デバイスが上記の要件を満たしていない場合は、Lingmo OSの軽量版をダウンロードできます。
