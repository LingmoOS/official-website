Lingmo OS

Debian GNU/Linuxに基づくモダンなLinuxディストリビューション

Lingmo OSは、安定したDebian GNU/Linuxに基づいており、エレガントなユーザーインターフェイス、安定したシステムパフォーマンス、豊富なソフトウェアエコシステムを提供し、あなたのLinux体験をよりシンプルでスムーズにします。

システム機能

Lingmo OSはシステム全体を再設計し、視覚効果を最適化し、ユーザー体験を向上させました。

セキュリティと安定性

Lingmo OSは、非常に安定したDebian GNU/Linuxに基づいており、美しく信頼性の高いシステムです。

美しいインターフェイス

LingmoDEを採用しており、macOSに似たユーザー体験を提供し、クリーンで美しいインターフェイスとスムーズな操作性を実現しています。

ユーザー友善のツール

ビルトインのシステム設定とソフトウェアストアにより、システム管理とソフトウェアインストールが簡単で直感的になります。

豊富なソフトウェアエコシステム

Debianソフトウェアソースをサポートし、あらゆる種類のユーザーのソフトウェアニーズを満たす広大なLinuxアプリケーションにアクセスできます。

新しいグローバルメニューデザイン

Lingmo OSは、画面スペースを節約しながら多くの機能を提供するトップグローバルメニューデザインを採用しています。

システム要件

最高の体験を得るためには、デバイスが以下の要件を満たしていることを確認してください

プロセッサ

  • • 最低要求：Intel/AMDのデュアルコアプロセッサ
  • • 推奨構成：Intel Core i3/AMD Ryzen 3以上
  • • x86_64アーキテクチャのみをサポート

メモリ

  • • 最低要求：2 GB RAM
  • • 推奨構成：4 GB RAM以上
  • • SWAPパーティションを有効化することをお勧めします

ストレージ容量

  • • 最低要求：10 GBの空き容量
  • • 推奨構成：30 GB以上の空き容量
  • • ext4、btrfsファイルシステムをサポート

グラフィックカード

  • • Intel HD Graphics 4000以上をサポート
  • • NVIDIAグラフィックカードをサポート（オープンソースドライバーをお勧め）
  • • AMD Radeonグラフィックカードをサポート

ネットワーク

  • • インストールと更新にはインターネット接続が必要です
  • • 10 Mbps以上のブロードバンド接続をお勧めします
  • • Wi-Fiと有線ネットワーク接続をサポート

その他の要件

  • • UEFIファームウェアサポート（推奨）
  • • TPM 2.0セキュリティチップ（オプション）

インストールガイド

Lingmo OSをデバイスに簡単にインストールするには、以下の手順に従ってください

1

インストールメディアの準備

システムイメージをダウンロードし、起動可能なUSBドライブを作成します

2

ブート順序の設定

BIOSに入り、USBドライブからのブートを設定します

3

インストールの完了

インストールウィザードに従ってシステムインストールを完了します

1
インストールメディアの準備

  1. Lingmo OSシステムイメージファイル（.iso）をダウンロードします
  2. 8GB以上の容量があるUSBドライブを用意してください（注意：USBドライブのデータは消去されます）
  3. balenaEtcherをダウンロードしてインストールします
  4. ツールを実行し、ダウンロードしたイメージファイルとUSBドライブを選択します
  5. 「開始」をクリックし、処理が完了するのを待ちます

ヒント：処理中にUSBドライブを抜かないでください。処理が完了したら安全に取り外してください。

2
ブート順序の設定

  1. 準備した起動可能なUSBドライブを挿入します
  2. コンピュータを再起動し、起動時に特定のキーを押してBIOS設定に入ります（一般的なキー：F2、F10、F12、DEL）
  3. BIOSで、「ブート順序」または「ブートシーケンス」オプションを見つけます
  4. USBドライブを最初のブートデバイスに設定します
  5. 設定を保存し、BIOSを退出します

BIOSに入るキーは、コンピュータのブランドによって異なる場合があります。コンピュータのマニュアルを参照してください。

3
インストールの完了

  1. コンピュータはUSBドライブから起動し、Lingmo OSインストールプログラムに入ります
  2. 言語と地域設定を選択します
  3. ユーザー名とパスワードを設定します
  4. インストール場所（ディスクパーティション）を選択します
  5. インストールが完了するのを待ちます
  6. 最初の起動では、プロンプトに従って初期設定を完了します

警告：新しいインストールでは、選択したパーティションのすべてのデータが消去されます。重要なファイルを事前にバックアップしてください。

Lingmo OSはDebian GNU/Linux Stableに基づいています。

これは通常、ネットワークの問題が原因です。以下のことができます：

  2. この段階でシステムはすでにインストールされているため、直接システムを再起動することができます。
