System Requirements

Ensure your device meets the following requirements for the best experience.

Processor

  • • Minimum Requirement: Intel/AMD Dual-core Processor
  • • Recommended Configuration: Intel Core i3/AMD Ryzen 3 or higher
  • • Only supports x86_64 architecture

Memory

  • • Minimum Requirement: 2 GB RAM
  • • Recommended Configuration: 4 GB RAM or higher
  • • It is recommended to enable SWAP partition

Storage Space

  • • Minimum Requirement: 10 GB available space
  • • Recommended Configuration: 30 GB or more available space
  • • Supports ext4, btrfs file systems

Graphics Card

  • • Supports Intel HD Graphics 4000 and above
  • • Supports NVIDIA graphics cards (open-source drivers recommended)
  • • Supports AMD Radeon graphics cards

Network

  • • Internet connection required for installation and updates
  • • Recommended broadband connection (10 Mbps or higher)
  • • Supports Wi-Fi and wired network connections

Other Requirements

  • • UEFI firmware support (recommended)
  • • TPM 2.0 security chip (optional)

Ensure Your Device Meets the Above Requirements

If your device does not meet the above requirements, you can download the lightweight version of Lingmo OS.
