System Requirements

Ensure your device meets the following requirements for the best experience

Processor

  • • Minimum Requirement: Dual-core processor from Intel/AMD
  • • Recommended Configuration: Intel Core i3/AMD Ryzen 3 or higher
  • • Only supports x86_64 architecture

Memory

  • • Minimum Requirement: 2 GB RAM
  • • Recommended Configuration: 4 GB RAM or higher
  • • It is recommended to enable a SWAP partition

Storage Space

  • • Minimum Requirement: 10 GB available space
  • • Recommended Configuration: 30 GB or more available space
  • • Supports ext4, btrfs file systems

Graphics Card

  • • Supports Intel HD Graphics 4000 and above
  • • Supports NVIDIA graphics cards (open-source drivers recommended)
  • • Supports AMD Radeon graphics cards

Network

  • • Internet connection required for installation and updates
  • • Recommended broadband connection (10 Mbps or higher)
  • • Supports Wi-Fi and wired network connections

Other Requirements

  • • UEFI firmware support (recommended)
  • • TPM 2.0 security chip (optional)

Complete Installation

Follow the installation wizard to complete the system installation

1
Prepare Installation Media

  1. Download the Lingmo OS system image file (.iso)
  2. Prepare a USB drive with at least 8GB capacity (note: data on the USB drive will be erased)
  3. Download and install balenaEtcher
  4. Run the tool, select the downloaded image file and the USB drive
  5. Click "Start" and wait for the process to complete

Tip: Do not remove the USB drive during the process. Safely eject it after completion.

2
Set Boot Order

  1. Insert the prepared bootable USB drive
  2. Restart your computer and press a specific key during startup to enter BIOS settings (common keys: F2, F10, F12, DEL)
  3. In the BIOS, find the "Boot Order" or "Boot Sequence" option
  4. Set the USB drive as the first boot device
  5. Save the settings and exit the BIOS

The key to enter BIOS may vary depending on the brand of your computer. Please refer to your computer's manual.

3
Complete Installation

  1. The computer will boot from the USB drive and enter the Lingmo OS installation program
  2. Select language and region settings
  3. Set up a username and password
  4. Choose the installation location (disk partitioning)
  5. Wait for the installation to complete
  6. During the first boot, follow the prompts to complete the initial setup

Warning: A fresh installation will erase all data on the selected partition. Please back up important files in advance.
