系统要求

确保您的设备满足以下要求，以获得最佳体验

处理器

  • • 最低要求: Intel/AMD 双核处理器
  • • 推荐配置: Intel Core i3/AMD Ryzen 3 或更高
  • • 仅支持 x86_64 架构

内存

  • • 最低要求: 2 GB RAM
  • • 推荐配置: 4 GB RAM 或更高
  • • 建议开启 SWAP 分区

存储空间

  • • 最低要求: 10 GB 可用空间
  • • 推荐配置: 30 GB 或更多可用空间
  • • 支持 ext4, btrfs 文件系统

显卡

  • • 支持 Intel HD Graphics 4000 及以上
  • • 支持 NVIDIA 显卡（推荐使用开源驱动）
  • • 支持 AMD Radeon 显卡

网络

  • • 安装和更新需要互联网连接
  • • 推荐宽带连接 (10 Mbps 或更高)
  • • 支持 Wi-Fi 和有线网络连接

其他要求

  • • UEFI 固件支持 (推荐)
  • • TPM 2.0 安全芯片 (可选)

安装指南

按照以下步骤，轻松安装 Lingmo OS 到您的设备

完成安装

按照安装向导完成系统安装

1
准备安装介质

  1. 下载 Lingmo OS 系统镜像文件 (.iso)
  2. 准备一个至少 8GB 容量的 U 盘 (注意：U 盘数据将被清除)
  3. 下载并安装 balenaEtcher
  4. 运行工具，选择下载的镜像文件和 U 盘
  5. 点击"开始制作"，等待完成

提示：制作过程中请勿拔出 U 盘，完成后请安全弹出。

2
设置启动顺序

  1. 插入制作好的启动 U 盘
  2. 重启计算机，在开机画面按下特定键进入 BIOS 设置 (常见键：F2, F10, F12, DEL)
  3. 在 BIOS 中找到"启动顺序"或"Boot Order"选项
  4. 将 U 盘设置为第一启动项
  5. 保存设置并退出 BIOS

不同品牌的电脑进入 BIOS 的按键可能不同，请参考您的电脑说明书。

3
完成安装

  1. 电脑将从 U 盘启动，进入 Lingmo OS 安装程序
  2. 选择语言和区域设置
  3. 设置用户名和密码
  4. 选择安装位置 (硬盘分区)
  5. 等待安装完成
  6. 首次启动时，按照引导完成初始设置

警告：全新安装将清除所选分区的所有数据，请提前备份重要文件。
